Некоторые правила тестирования
1. Увас будет 15 секунд для ответа на вопрос.
2. Выбранный вариант ответа нельзя изменить.
3. По истечению времени, данному для ответа, ответить на вопрос нельзя (вопрос не засчитывается).
4. Нельзя выйти из теста, не дойдя до последнего вопроса.
5. Результат рассчитывается исходя из количества правильных ответов.
